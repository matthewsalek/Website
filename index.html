Internships

Behold Control

Embedded Software Developer

Behold Control is a building automation company, making a smart thermostat for large buildings. I joined Behold as the first full time employee. In fact, my first day there was only one desk and one chair present in the entire office. It was fantastic experience, starting a product from essentially the ground up. I was given full freedom to direct how the product would be developed, how the embedded system would be designed, and what features would be most valuable for customers. I designed and developed all aspects of the controller, and worked on aspects such as the back end and front end of the web application, encryption and current sensors. By having so much to do in so little time at Behold, I gained valuable skills in project management, time management and communication.

Mercator Robotec

Mechanical Design Engineer

Mercator Robotec creates machine vision solutions for customers. At Mercator, I designed components for a robotic system, capable of detecting defects of needles commonly used in hospitals. Design Project - Optimum Angle of Attack for Airfoil

Ansys (Fluent)

I worked with a team of 4 in order to model an airfoil efficiency at differing angles of attack in order to find the angle which is most optimal. Staying up all night running apprixamately 20 computers in parallel produced the quality of data we needed. I gained knowledge in finite element analysis, using Ansys for engineering application, and aerodynamics.

Real Time System Multithreaded Sorting

C

With this project I implemented a sorting implementation for a real time system. I used quicksort on an array given, and at small array sizes, switched the sorting alogirthm in order to increase the speed of the application. I created a new thread for the left and right hand of the pivot, and used semaphores in order to protect a critical region that involved incrementing and decremting a counter. I gained fundamental understanding of aspects of a real time system.

Static Truss Structure Analysis

MATLAB

Contributed to group project consisting of analayzing truss structure. The result was that given any 2D truss structure that is determinant, the software written will be able to solve for all forces and displacements.

Dynamic Truss Structure Analysis

MATLAB

The next level of project 4, I implemented explicit dynamics for solving a dynamic truss structure. This generic algorithm was then applied to solve two dynamic systems, one dealing with a truss, and the other dealing with a beam.
